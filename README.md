# maths-quiz
maths quiz for km to m converter

currently using is hero_monster_v3
Features: 
1. it is a battle arena which player fight monsters by answering the maths questions
2. after a monster defeated, the player health point will restore a bit, and a stronger monster with longer health point and higher damage will appear.
3. the level of the hero remained unchanged, while the level of the monster will become higher and higher throughout the game.
   
Improvement made from previous version:
1. unnecessary zeros alert is added to the game
2. more random question format is generated

rocket is also an interesting python script game as it will show exactly how far the scale the rocket flew entered by the player.
